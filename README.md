# Copiloto de Vendas com IA para Loja de Ferramentas

## Sobre o projeto

Este projeto apresenta um Copiloto de Vendas com IA voltado para uma loja de ferramentas, EPIs, máquinas, acessórios e produtos para manutenção, construção e uso profissional.

A ideia é ajudar vendedores a entender melhor a necessidade do cliente, sugerir boas perguntas, recomendar produtos adequados e criar respostas prontas para atendimento em WhatsApp, balcão ou telefone.

## Problema que o projeto resolve

Em uma loja de ferramentas, muitos clientes chegam pedindo um produto sem saber exatamente qual modelo, tamanho, potência ou acessório precisam.

Exemplos:

- Cliente quer uma furadeira, mas não sabe se precisa com impacto;
- Cliente quer uma luva, mas não sabe qual material é adequado;
- Cliente quer cortar ferro, mas não sabe qual disco usar;
- Cliente quer o produto mais barato, mas pode acabar comprando algo inadequado;
- Cliente esquece EPIs ou acessórios importantes para usar a ferramenta com segurança.

O copiloto ajuda o vendedor a evitar indicações erradas e melhorar o atendimento.

## Abordagem utilizada

A abordagem utilizada foi a de Copiloto de Vendas com IA.

O copiloto não substitui o vendedor. Ele atua como apoio, sugerindo:

- Leitura do interesse do cliente;
- Diagnóstico de oportunidade;
- Perguntas de qualificação;
- Oferta principal recomendada;
- Oferta complementar inteligente;
- Estratégia de ancoragem;
- Mensagem pronta para o vendedor enviar ao cliente.
